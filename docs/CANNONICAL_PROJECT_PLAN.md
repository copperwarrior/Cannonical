# Cannonical Project Plan

## Project Scope
Cannon mod for the Shipwrights Naval Suite.

## Cannon Basics
- Heavily inspired by Sea of Thieves
- Highly aimable
- Easy but slow to reload
- Can fire players

## Cannon Types

### 4 Pounder Pivot Gun
- 180 rotation cone
- Limited range
- Fast reload

### 24 Pounder Gun
- 45 yaw, 90 pitch
- Regular range
- Regular reload
- Standard cannon

### 32 Pounder Gun
- 30 yaw, 45 pitch
- Long range
- Slow reload

## Ammo Types
- Cannonball
- Grape shot
- Chain shot
- Flame shot
- Special cannonballs (ballast ball, helm ball, peace ball)

## Additional Explosive Utility
- Gunpowder barrel that can receive multiple different item inputs before being closed off, used as a multipurpose explosive
